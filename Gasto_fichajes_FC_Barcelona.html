ANLISIS DEL IMPACTO DEL GASTO EN FICHAJES DEL FC BARCELONA (2013 -2023)

PASO 1: Definimos el objetivo

En este caso vamos a analizar si el gasto en fichajes del FC Barcelona entre 2013 y 2023 ha generado un impacto positivo en el rendimiento deportivo ( ttulos, posicin en liga, etc..)

PASO 2: Conseguir y preparar los datos

- Qu datos vamos a necesitar?

  • 1.- Temporada
  • 2.- Gasto en fichajes (entradas)
  • 3.- Ingreso por ventas (salidas)
  • 4.- Balance neto
  • 5.- Ttulos ganados
  • 6.- Posicin en liga
  • 7.- Rendimiento en Champions

# Vamos a probar con una regresin lineal
from sklearn.linear_model import LinearRegression
import numpy as np
# Variables independientes y dependientes
X = df[["Net Spend (M)"]]
y = df [["Total Titles"]]

# Creamos y entrenamos el modelo
model = LinearRegression()
model.fit(X, y)

# Predicciones
y_pred = model.predict(X)

# Grfico de regresin
plt.figure(figsize=(8,5))
plt.scatter(X, y, color = 'blue', label = 'Datos reales')
plt.plot(X, y_pred, color = 'red', label = 'Lneas de regresin')
plt.title("Regresin lineal: inversin vs titulos")
plt.xlabel("Net Spend (M)")
plt.ylabel("Ttulos ganados")
plt.legend()
plt.tight_layout()
plt.show()

PASO 6: Conclusiones

  • Gasto mximo: La temporada 2017/18 con 375M gastados.
  • Mayor rendimiento deportivo: Fue en la temporada 2014/2015 que obtuvieron el triplete con un gasto moderado.
  • Correlacin dbil entre inversin y ttulos: Gastar ms no garantiza ganar ttulos
  • Mejor relacin coste/rendimiento: Temporadas con plantillas ms equilibradas y menos gasto.
